# DX検定Pass プライバシーポリシー

最終更新日: 2026年7月4日

DX検定Pass（以下「本アプリ」）は、DX Next検定およびDXビジネス検定の学習機能、広告表示、アプリ内課金、利用状況の集計を提供するために、端末上の学習履歴および外部サービスが提供する識別子等を利用する場合があります。

本アプリは、ログインやアカウント作成を必要としません。

## 収集する情報

- 学習履歴、回答履歴、正誤履歴、苦手問題、連続学習日数など、アプリ内で保存される学習データ
- アプリの設定情報
- 広告配信、広告効果測定に必要な情報
- アプリ内課金の購入状態を管理するために必要な情報
- 利用状況の把握に必要な技術情報

本アプリは、氏名、住所、電話番号、メールアドレス、連絡先、正確な位置情報を、アカウント情報として独自に取得しません。

本アプリは、勤務先名、所属部署、取引先情報、社内資料、業務上の機密情報、個別企業のDX計画、自由記述の答案を入力・保存・送信する機能を持ちません。

## 利用目的

- 問題演習、復習、学習進捗表示の提供
- 誤答優先学習、苦手問題、連続学習日数、実戦演習などの学習機能の提供
- 広告表示
- アプリ内課金の購入状態の確認、復元、管理
- 品質改善、利用状況の分析

## 学習データの保存

学習履歴、回答履歴、苦手問題、連続学習日数などの学習データは、原則として利用者の端末内に保存されます。

本アプリは、学習履歴を管理するための独自アカウントサーバーを運営していません。端末の削除、アプリの削除、端末変更などにより、端末内の学習データが失われる場合があります。

## 広告について

本アプリでは、Google Mobile Ads を利用して、バナー広告、インタースティシャル広告、リワード広告を表示する場合があります。

Google Mobile Ads では、広告配信、広告効果測定、不正防止などのために、端末情報、広告識別子、利用状況に関する情報が利用される場合があります。これらの情報の取り扱いは、Google のポリシーに従って管理されます。

リワード広告は、利用者が出題設定画面で広告視聴ボタンを押した場合にのみ表示します。広告視聴が完了した場合、選択した分野の追加問題を一時的に解放します。広告の読み込み失敗、表示失敗、途中終了、タイムアウトの場合は追加解放を行いません。

## Firebase Analytics について

本アプリでは、利用状況の把握と学習体験の改善のため、Firebase Analytics を利用します。

Firebase Analytics では、アプリ起動、学習開始、回答、学習完了、実戦演習開始、実戦演習完了、購入画面表示、購入開始、購入完了、レビュー依頼候補などのイベントを、個人を直接特定しない集計用情報として送信する場合があります。

本アプリは、Firebase Analytics に問題本文、選択肢本文、解説本文、question_idの生値、氏名、メールアドレス、自由入力、個人情報、外部サービスの内部識別子、決済識別子の生値を送信しません。

本アプリでは、Firebase Analytics を広告識別子を利用しない構成で使用し、App Tracking Transparency の許可を求める機能は実装していません。

本アプリでは、Firebase Authentication、Firestore、Realtime Database、Functions、Storage、Remote Config、Crashlytics、Cloud Messaging、App Check は利用しません。

## アプリ内課金について

本アプリでは、非消耗型のアプリ内課金により、Premium 機能を提供する場合があります。

Premium では、全問題の利用、広告非表示、詳しい解説、苦手問題モード、実戦演習などの機能を利用できます。

購入処理は Apple の仕組みを通じて行われ、購入状態の管理には RevenueCat を利用します。RevenueCat では、購入状態の確認、復元、課金機能の提供に必要な識別子や購入情報が取り扱われる場合があります。

## 第三者サービス

本アプリでは、広告配信、課金管理、利用状況の分析のために、以下の第三者サービスを利用する場合があります。各サービスで取り扱われる情報は、それぞれのプライバシーポリシーに従って管理されます。

- Google Mobile Ads
- Firebase Analytics
- RevenueCat
- Apple が提供する App Store、アプリ内課金、診断関連サービス

## 外部送信

広告配信、課金管理、利用状況の分析、品質改善のため、第三者サービスに端末情報、広告関連情報、購入情報、診断情報などが送信される場合があります。

本アプリは、利用者が入力した氏名、住所、電話番号、メールアドレス、連絡先、正確な位置情報、勤務先名、所属部署、取引先情報、社内資料、業務上の機密情報、個別企業のDX計画、自由記述の答案を独自に収集して外部送信する機能を持ちません。
